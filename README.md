SPECTRA v3 — Forensic Audio Editor
SPECTRA v3 is a professional forensic audio enhancement tool designed for isolating whispered conversations and cleaning up difficult recordings. Built for investigators and audio analysts who need precision tools without expensive software.
Load any audio file and immediately see your problem frequencies lit up in real time across a color-coded spectrum analyzer. Dedicated diagnostic meters track room hum, preamp static, voice presence, reverb tail, and noise floor simultaneously, giving you a live Voice Clarity Score as you work.
Clean up your audio using an 8-band parametric EQ tuned to the human voice range, surgical notch filters for 60/120Hz hum, spectral subtraction noise profiling, dereverberation, and a soft noise gate. One-click presets for Whisper Enhance, Room Cleanup, Preamp Denoize, and Voice Isolate get you started fast.
Navigate with precision using the haptic scrub dial — drag forward or backward in 10ms increments while your phone vibrates on each tick. The magnascope window shows a zoomed waveform around your cursor down to millisecond accuracy. Export your cleaned recording as a WAV file when done.
Installs as a Progressive Web App directly to your home screen. No app store, no subscription, no internet required after install.
